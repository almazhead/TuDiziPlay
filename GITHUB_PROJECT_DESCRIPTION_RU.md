# TuDiziPlay

TuDiziPlay — это Android-приложение для просмотра турецких сериалов, серий и последних обновлений в лёгком и понятном интерфейсе.

Приложение разработано для Android-смартфонов, адаптировано для планшетов и оптимизировано для Android TV. Основной акцент сделан на быстрой навигации, удобном выборе серий и комфортном использовании на разных размерах экрана.

## Основные возможности

- Удобный просмотр каталога
- Быстрый поиск и отслеживание обновлений
- Комфортный выбор серий и навигация по сериалу
- Интерфейс, адаптированный для телефона, планшета и Android TV

## Технологии

TuDiziPlay создан на Flutter и Dart для Android-устройств. Проект включает адаптивный интерфейс для телефонов и планшетов, а также отдельную оптимизацию для Android TV.
